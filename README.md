# Fitness SC - Gym Management System

A modern, responsive gym management website built with React and Vite. Features include a dynamic landing page, service listings, trainer profiles, subscription plans, and a contact section.

## ● Project Name
**Fitness SC - Gym Management System**

## ● Tech Stack
- **Frontend**: [React 19](https://react.dev/)
- **Build Tool**: [Vite](https://vite.dev/)
- **Styling**: Vanilla CSS (Custom properties for Light/Dark modes)
- **Features**: Scroll Reveal Animations, Responsive Design, Theme Switching

## ● Setup Instructions

### Prerequisites
- [Node.js](https://nodejs.org/) (v18 or higher recommended)
- npm (comes with Node.js)

### Installation Steps
1. **Clone the repository**:
   ```bash
   git clone <repository-url>
   ```
2. **Navigate to the project directory**:
   ```bash
   cd gym
   ```
3. **Install dependencies**:
   ```bash
   npm install
   ```
4. **Start the development server**:
   ```bash
   npm run dev
   ```
   The application will be available at `http://localhost:5173`.

### Building for Production
To create an optimized production build, run:
```bash
npm run build
```
The output will be in the `dist` directory.
